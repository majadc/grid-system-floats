The grid system - based on floats


The grid is based on 12 columns in a row.


<div class="grid-container">
  <div class="row">
    <div class="column-n">Column</div>
    <div class="column-n">Column</div>
  </div>
</div>

n - the number of column-elements (form 1 to 12. The summ must be 12)


<div class="grid-container">
  <div class="row">
    <div class="column-6">Column</div>
    <div class="column-6">Column</div>
  </div>
</div>

The column width is a multiple of the width of the column-element (column-1). 


	[class*="column-"] {
		float:left;
	}
	.column-1 {
		width: calc(100%/12); /*  width: 8.3333333333%; */
	}
	.column-6 {
		width: width: 50%; /*  6 * 8.3333333333%; */
	}
